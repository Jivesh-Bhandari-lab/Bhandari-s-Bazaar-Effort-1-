# Bhandari's Market 🛍️

**"Bringing the Heart of the Bazaar Online"**

A beautiful, modern, fully functional **local online marketplace** built with pure HTML, Tailwind CSS, and Alpine.js. It simulates a real e-commerce platform connecting local sellers, artisans, and farmers directly with customers.

## ✨ Features

### For Customers:
- **Stunning Landing Page** with fresh produce focus (Basic codes used)
- **Category Browsing**: Clothes, Accessories, Groceries, Electronics
- **Interactive Product Pages** with image carousels and filters
- **Smooth Shopping Experience** with "Buy Now" simulation
- **Order History** page with status tracking
- **User Profile** with editable details and avatar upload
- **Login / Registration** portal with localStorage auth simulation

### For Sellers:
- **Seller Dashboard** ("My Products")
- **Add New Product** modal with image upload
- **Manage Inventory** (add/remove listings)

### Technical Highlights:
- Fully responsive design (mobile + desktop)
- Tailwind CSS + Alpine.js (no heavy frameworks)
- Interactive carousels, filters, and modals
- LocalStorage-based persistence (cart simulation, user data)
- Clean, modern UI inspired by premium marketplaces

## 🛠️ Technologies Used

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Alpine.js** (for reactivity)
- **Google Fonts** (Inter)
- **Unsplash** images (placeholder)
